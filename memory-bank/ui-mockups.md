## Key UI Design Elements

### Dropbox Design Style Features:
- **Flat Design**: No 3D effects, gradients, or shadows
- **Thick Black Borders**: All buttons and form elements have prominent borders
- **Transparent Backgrounds**: Buttons show background color through transparency
- **Bold Typography**: Clear, readable text hierarchy
- **Vibrant Colors**: Strategic use of bright colors for CTAs and status indicators
- **Minimal Visual Clutter**: Clean, spacious layouts with plenty of whitespace

### Functional Elements:
1. **Client-side Templates**: Pre-configured experiment types that prefill forms locally
2. **Experiment Status**: Status display for completed experiments
3. **Economic Impact Focus**: Dollar values prominently displayed throughout
4. **Data Visualization**: Simple ASCII charts for trend analysis
5. **Metric Configuration**: Easy management of economic value mappings
6. **Export/Share**: Data portability for reporting and collaboration

### Navigation Flow:
- Dashboard → Create Experiment → View Results → Manage Metrics
- Each page provides clear navigation back to dashboard
- Simple workflow: Create → Auto-complete → View Results
